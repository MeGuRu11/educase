# EpiCase — вопросы к кафедре

Здесь остаются только внешние решения и материалы, которых нет в коде или принятых ADR.
Инженерные вопросы перенесены в архитектурные решения и `DATA_MODEL.md`.

## Решено

- [x] Две программы, файловый обмен и отсутствие сети — ADR-001..003.
- [x] Шесть фиксированных этапов и неблокирующее ветвление — ADR-004/005.
- [x] Строгий поиск по синонимам преподавателя — ADR-006.
- [x] Формат ZIP, manifest, checksum и ассеты — ADR-009/010 и FEATURE-01.
- [x] Схема объекта «фон + прямоугольные хотспоты» — ADR-013.
- [x] Документы заполняются через `FIELDS` или `ATTACHMENT` — ADR-015.
- [x] Машинная проверка даёт подробную подсказку без балла/pass-fail; окончательное
  решение принимает преподаватель — ADR-016.
- [x] Фактическая модель `Case`/`Attempt`/`Report` описана в `DATA_MODEL.md`.
