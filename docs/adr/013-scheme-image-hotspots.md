# ADR-013: Схема объекта — «фон + хотспоты» вместо векторного редактора

## Статус
Принят. **Отменяет ADR-012** (полноценный векторный редактор полигонов/стен/слоёв).

## Контекст
ADR-012 предполагал встроенный векторный редактор схем (рисование помещений
вершина-за-вершиной, стены, слои), запланированный последней и самой дорогой фичей.

Разбор исходного спец-эскиза кафедры (СХЕМА.xlsx) показал, что под «схемой» понимается
не точный чертёж, а: «общая схема объекта» (изображение) с «ячейками под личный состав»,
«иконками», «фото по объектам внешней среды», «блоком иконок действий» и диалогами
«что хотите сделать?», ведущими к контекстному поиску и документам. Это подтверждает
механика «виртуального обхода»: план → кликабельные входы → интерьерные фото с
хотспотами → свободный вывод по ключевым словам.

То есть преподаватель предоставляет общую схему готовым изображением и расставляет поверх
интерактивные элементы. Векторное рисование с нуля не требуется.

## Решение
Схема — это `SchemeDocument`: рекурсивные `SchemeView` (фоновое изображение по id ассета +
прямоугольные `Hotspot` в нормализованных координатах [0..1]). Хотспот при клике раскрывает
текст и/или ассеты (как `SearchEntry`) либо открывает вложенный `SchemeView` (интерьерный
вид со своими хотспотами). Координаты — доли, чтобы не зависеть от размера изображения и зума.

Проверка осмотра остаётся прежней — `InspectionCheck` (сверка свободного вывода курсанта по
покрытию групп). Схема отвечает только за навигацию и раскрытие, не за оценивание.

Никакого рисования полигонов, стен и слоёв. Редактор в Constructor — размещение
прямоугольников поверх загруженного изображения (`QGraphicsScene`), не CAD.

## Последствия
- Режется самый дорогой кусок проекта; модель переиспользует ассеты, `SynonymSet`,
  `InspectionCheck` — нового кода минимум.
- `attempt.py` и `report.py` не меняются (осмотр и его сверка прежние).
- В `StageContacts`/`StageEnvironment` поле `scheme: str | None` становится
  `SchemeDocument | None` — слом формата. Допустимо: архивов в обращении нет,
  `FORMAT_VERSION` остаётся 1, эталонный кейс ещё не получен.
- UI-редактор хотспотов на `QGraphicsScene` (масштаб/перетаскивание/hit-test) — фаза R2.
- Фазы: R1 — домен + просмотрщик; R2 — редактор размещения хотспотов; R3 — шлифовка.
