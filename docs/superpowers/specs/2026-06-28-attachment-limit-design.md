# Лимит вложений документа

## Статус

Утверждено пользователем 28 июня 2026 года.

## Цель

Каждый блок документа в режиме `ATTACHMENT` позволяет курсанту прикрепить до 10 файлов.
Поведение одинаково для новых и ранее созданных `.epicase`.

## Совместимость

- Форматы `.epicase` и `.epiresult` и их версии не меняются.
- Поле `allow_multiple` остаётся в доменной и документной модели для чтения старых архивов.
- Player больше не использует `allow_multiple`: любой блок `ATTACHMENT` принимает несколько
  файлов, но не более 10.
- Constructor больше не показывает настройку «Можно прикрепить несколько файлов». Для новых
  и повторно сохранённых шаблонов `ATTACHMENT` он записывает совместимое значение
  `allow_multiple=True`.

## Player

- Кнопка называется «Прикрепить файлы» и открывает множественный выбор.
- Заголовок списка показывает `Прикреплённые файлы · N / 10`.
- Каждый файл остаётся отдельной карточкой с удалением по одному.
- Кнопка «Очистить всё» удаляется.
- При 10 вложениях кнопка прикрепления отключается; удаление карточки снова её включает.
- Если пользователь выбрал больше файлов, чем осталось свободных мест, добавляются только
  первые доступные файлы и показывается нейтральное сообщение
  «Можно прикрепить не более 10 файлов».
- Лимит применяется отдельно к каждому `DocumentWidget`; содержимое файлов и повторяющиеся
  имена не анализируются.

## Constructor и отчёт

- В редакторе шаблона удаляется переключатель количества вложений.
- Отчёт продолжает показывать все сохранённые вложения карточками без изменений формата.

## Документация и тесты

- ADR-015 и проектный skill документов синхронизируются с единым лимитом.
- Тесты покрывают старый шаблон с `allow_multiple=False`, добавление до лимита, отсечение
  лишних файлов, отключение/повторное включение кнопки и отсутствие «Очистить всё».
- После изменения выполняется полный project quality gate.

## Живая проверка

1. В старом `shigellosis.epicase` на этапе 4 прикрепить несколько файлов за один выбор.
2. Добавить файлы повторным выбором до общего количества 10.
3. Убедиться, что одиннадцатый файл не добавляется, а кнопка прикрепления отключена.
4. Удалить одну карточку и убедиться, что снова можно прикрепить один файл.
5. Сохранить `.epiresult` и проверить в Constructor все вложения блока.
