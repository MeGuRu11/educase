# EduCase — проектный скилл

## Что за проект
Тренажёр для подготовки военных эпидемиологов (ВМА им. Кирова), Windows 10/11, компьютерный
класс. **Две раздельные программы:** Constructor (преподаватель, без авторизации) и Player
(курсант). Обмен — только файлами `.epicase`/`.epiresult` вручную. **Сети нет (ADR-003).**
Дедлайн 25 июня 2026. Принцип: скорость до работающего кода, без раздувания.

## Архитектура
Общий `epicase_core` (domain → application → infrastructure) + два GUI-приложения.
Персистентность — только кодек архива `.epicase`/`.epiresult` (ADR-009), БД/ORM нет.
Запрещено: domain ← PySide6/внешние библиотеки; UI ← infrastructure напрямую; любой код ← сеть.

## Quality gate (после каждого изменения)
`ruff check src tests` → `mypy src tests` → `pytest -q` → `python -m compileall -q src tests`
→ коммит (Conventional Commits).

## Жёсткие правила
- Сети нет (ADR-003). JSON не показывается пользователю. Шесть фиксированных этапов.
- Неверные ответы не блокируют прохождение (вскрываются в финальном отчёте).
- Поиск по ключевым словам строгий: синонимы от преподавателя, без fuzzy/опечаток.
